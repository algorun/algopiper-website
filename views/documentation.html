Follow these steps to create a workflow!

STEP 1: Deploy AlgoPiper

From the top navigation bar, choose Try Now!. Hit the Deploy button. This will open a new tab with an isolated AlgoPiper environment for you (allow popups when asked). The main AlgoPiper interface looks like:

STEP 2: Drag AlgoRun Node and Choose an Algorithm

The left sidebar shows a number of available nodes that can be added to a workflow. Drag an AlgoRun node into the workspace. Double click a node to choose an algorithm. When you submit algorithms to AlgoRun, it will be automatially available in the list









STEP 3: Read Algorithm Information

On the right sidebar, the info tab will show detailed information about the algorithm and its usage. Make sure to check the input and output format to be able to build a workflow correctly. Check our list of input and ouptput types.

STEP 4: Add Input and Output Nodes

Input nodes are used to start the workflow computation. Edit the input node by adding your input in the text box. Output nodes are used to print the output of the computation to the right sidebar (OUTPUT tab).



STEP 5: Connect Nodes

Connect the right dot of the input node to the left dot of the AlgoRun node. Connect the right dot of the AlgoRun node to the left dot of the output node.

STEP 6: Deploy and Change Parameters

On the top right corner, hit deploy. AlgoRun node status will change to ready. On the right sidebar, you can change the parameters of this node from the Parameters tab.

NOTE: In some browsers, you may need to refresh the page after you hit deploy for the node status (e.g. the green ready) to show up. This doesn't affect the operation of the workflow though.



STEP 7: Start the Computation

Hit the button on the left of the input node to trigger the workflow. This will inject the input data to the AlgoRun node. The AlgoRun node will automatically perform the computation and passes the output to the next node; which is the output node in this simple workflow. The output node will print the output to the right sidebar. Openning the log file of the AlgoRun node helps troubleshoot complex workflows.





STEP 8: Export Workflows

From the top right corner, export the workflow to a local file for future use. Using AlgoPiper through AlgoRun website is free but the session is limited to 24 hours, then your isolated environment will be premenantly delelted. You can start a new session (when the first one is done) and import your saved workflow.

TIP: Submit a workflow from the top navigation bar to host your workflows permanently on our servers ..
