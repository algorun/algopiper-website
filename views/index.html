A visual tool to create modular pipelines from AlgoRun algorithms

What is AlgoPiper?

The problem: There is a growing need in bioinformatics (and computational science in general) for easy-to-use software tools that facilitates building and troubleshooting pipelines. At the same time, available tools requires that users pass through a lengthy process of local installations and configurations for individual modules. Testing mathematical models using pipelines becomes a hurdle for researchers.

The solution: AlgoPiper is a web-based drag-n-drop visual tool to create modular and reusable computational pipelines from AlgoRun algorithms. AlgoPiper makes troubleshooting complex pipelines a few clicks process. Build your first pipeline now through an interactive tutorial ..
